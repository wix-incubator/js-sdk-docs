<static> getAudioUrl(relativeUri, audioType) → {String}

Constructs an absolute URL for a relative path to an audio file. By default, returns a URL to a standard audio file.

Parameters:
Name Type Description
relativeUri String

A relative URL to the target audio file.
audioType Wix.Utils.Media.AudioType

the type of audio URL to build. Default is Wix.Media.AudioType.STANDARD.
Since:
  • 1.45.0
Source:
Media.js
Returns:

An absolute URL pointing to the audio file hosted on Wix's static.

Type
String
Example

<static> getResizedImageUrl(relativeUrl, width, height, sharpParams) → {String}

This method constructs a URL for a media item of type image and let the developer change the image dimensions as well as it's sharpening properties (optional), see sharpening explained here - http://en.wikipedia.org/wiki/Unsharp_masking.

Parameters:
Name Type Argument Description
relativeUrl String

Static image url provided by the media dialog.
width Number

Desired image width.
height Number

Desired image height.
sharpParams Object <optional>
Properties
Name Type Description
quality Number

JPEG quality, leave as is (75) unless image size is important for your app.
resizaFilter Number

Resize filter.
usm_r Number

Unsharp mask radius.
usm_a Number

Unsharp mask amount (percentage).
usm_t Number

Unsharp mask threshold.
Since:
  • 1.17.0
Source:
Media.js
Returns:

A full URL pointing to the Wix static servers of an image with the custom dimension parameters.

Type
String
Example
var resizedImageUrl = Wix.Utils.Media.getResizedImageUrl('relative_url.jpg', 500, 500)
