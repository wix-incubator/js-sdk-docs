getAudioUrl( )

getAudioUrl( )

Description

Constructs an absolute URL for a relative path to an audio file. By default, returns a URL to a standard audio file.

Parameters

relativeUri String A relative URL to the target audio file.
audioType AudioType the type of audio URL to build. Default is Wix.Media.AudioType.STANDARD.

Return Value

String An absolute URL pointing to the audio file hosted on Wix's static.

Examples

Availability

Since 1.45.0 and later.

getResizedImageUrl( )

getResizedImageUrl( )

Description

This method constructs a URL for a media item of type image and let the developer change the image dimensions as well as it's sharpening properties (optional), see sharpening explained here - http://en.wikipedia.org/wiki/Unsharp_masking.

Parameters

relativeUrl String Static image url provided by the media dialog.
width Number Desired image width.
height Number Desired image height.
sharpParams (optional) Object
quality Number JPEG quality, leave as is (75) unless image size is important for your app.
resizaFilter Number Resize filter.
usm_r Number Unsharp mask radius.
usm_a Number Unsharp mask amount (percentage).
usm_t Number Unsharp mask threshold.

Return Value

String A full URL pointing to the Wix static servers of an image with the custom dimension parameters.

Examples

Availability

Since 1.17.0 and later.
