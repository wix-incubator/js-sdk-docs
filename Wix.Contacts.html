getContactById( )

getContactById( )

Description

Gets a specific Contact that has interacted with the current site by its id.

Parameters

id String The id of the Contact to look up.
onSuccess function A function that receives data about the Contact.
onFailure function A function called when an error occurs that receives a Wix.Error.

Return Value

Contact

Availability

Since 1.27.0 and later.

getContacts( )

getContacts( )

Description

Gets a list of all contacts that have interacted with a given site.

Parameters

options Object object that supports two parameters: 'label' and 'pageSize'. 'label' can either be a list of strings or a string. if a list, the strings are joined together with a comma to be sent to the contacts endpoint. 'pageSize' accepts either 25, 50 or 100 and defaults to 25.
onSuccess function An on success callback which gets WixDataCursor as parameter.
onFailure function An on failure callback.

Return Value

WixDataCursor cursor.

Examples

Availability

Since 1.31.0 and later.

reconcileContact( )

reconcileContact( )

Description

Reconciles Contact information with that of the WixHive's.

Use this when your app has information about a site visitor that may already be registered as a Contact as part of the WixHive. Your app should provide as much information as possible so that we will find the best match for that Contact and return it with the reconciled information. If no match was found, we will create a new Contact.

Depending on the type of information, we will either add or dismiss changes. When the information can be added to a list, such as emails or phones, a new item will be added if no similar item exists. When the information cannot be added we dismiss the change, such is the case with name, company and picture. If your wish is to override such data, there are explicit ways to do so using the Contactâ€™s id and our REST API.

Parameters

contactInfo Object Contact information that will be passed to the server.
onSuccess (optional) function A function that receives reconciliation details.
onFailure (optional) function A function called when an error occurs, receives a Wix.Error.

Return Value

ReconcileContactResult Contains the reconciled Contact and details about the operation.

Availability

Since 1.46.0 and later.
