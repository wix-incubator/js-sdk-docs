Namespace: Billing

Wix. Billing

Functions and objects relating to the purchasing and billing process. To define products for purchase within your app, go to the Features section of the Wix Developer app registration dev.wix.com.

<static> getBillingPackages(vendorProductIds, onSuccess, onError) → {Array}

Returns an Array of objects containing product and pricing info. As is defined in the Features section of the Wix Developer App Registration dev.wix.com

Parameters:
Name Type Argument Description
vendorProductIds Array <optional>

A list of vendor product ids (each representing a billing package), as detailed in the Features section of the Wix Developer App Registration dev.wix.com
onSuccess function

A callback function to receive the product info.
onError function <optional>

A callback error function.
Since:
  • 1.37.0
Author:
  • lior.shefer@wix.com
Source:
Billing.js
Returns:

Array of Objects containing the product info:

Name Type Description
prices Array -
price Object Name Type Description
- - value String The product price.
- - currencyCode String The product payment currency code.
- - currencySymbol String The product payment currency symbol.
- - cycle Wix.Billing.Cycle The product payment cycle
Type
Array
Example
var onError = function () {
 //handle the error
};
var onSuccess = function (data) {
 //handle onSuccess
 //sample data schema:
 [{
    id: <vendorProductId>,
    prices: [
    {
      value : '3.99',
      currencyCode: 'USD',
      currencySymbol: '&#36;',
      cycle: 'MONTHLY'
    }]
 }]

};
Wix.Billing.getBillingPackages(onSuccess, onError);

<static> getBillingPageForProduct(vendorProductId, cycle, onSuccess, onError)

Returns a link to Wix Billing page with information about the product and cycle requested.

Parameters:
Name Type Argument Description
vendorProductId String

Vendor product id as detailed in the Features section of the Wix Developer App Registration dev.wix.com
cycle Wix.Billing.Cycle

The billing cycle.
onSuccess function

A callback function, returns a link the Wix billing page with information about the product and cycle requested.
onError function <optional>

A callback error function, Error is wrong cycle, missing cycle or bad productId.
Since:
  • 1.37.0
Author:
  • lior.shefer@wix.com
Source:
Billing.js
Example
var onError = function () {
 //handle the error
};
Wix.Billing.getBillingPageForProduct('vendorProductId', Wix.Billing.Cycle.MONTHLY, function () {
     //handle return value i.e., //https://premium.wix.com/...
}, onError);

<private, static> openBillingPageForProduct(vendorProductId, cycle, onError)
