getBillingPackages( )

getBillingPackages( )

Description

Returns an Array of objects containing product and pricing info. As is defined in the Features section of the Wix Developer App Registration dev.wix.com

Parameters

vendorProductIds (optional) Array A list of vendor product ids (each representing a billing package), as detailed in the Features section of the Wix Developer App Registration dev.wix.com
onSuccess function A callback function to receive the product info.
onError (optional) function A callback error function.

Return Value

Array Array of Objects containing the product info:
Name Type Description
prices Array -
price Object Name Type Description
- - value String The product price.
- - currencyCode String The product payment currency code.
- - currencySymbol String The product payment currency symbol.
- - cycle Wix.Billing.Cycle The product payment cycle

Examples

Availability

Since 1.37.0 and later.

getBillingPageForProduct( )

getBillingPageForProduct( )

Description

Returns a link to Wix Billing page with information about the product and cycle requested.

Parameters

vendorProductId String Vendor product id as detailed in the Features section of the Wix Developer App Registration dev.wix.com
cycle Cycle The billing cycle.
onSuccess function A callback function, returns a link the Wix billing page with information about the product and cycle requested.
onError (optional) function A callback error function, Error is wrong cycle, missing cycle or bad productId.

Availability

Since 1.37.0 and later.
