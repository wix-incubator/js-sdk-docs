publish( )

publish( )

Description

Broadcasts an event to other extensions of a multi-widget TPA. If the app extensions (Widget, Fixed Positioned Widget, Page) span multiple pages, they will be notified when they are rendered.

Parameters

eventName String The name of the event to publish.
data Object Data the object to send to subscribers for this event type.
isPersistent Boolean Indicates whether this event is persisted for event subscribers who have not yet subscribed.

Examples

Availability

Since 1.25.0 and later.

static

subscribe( )

subscribe( )

Description

Subscribes to events from other app parts of a multi-widget TPA. If the components span multiple pages, they will be notified once they are rendered. It is also possible to receive all notifications prior to rendering by specifying a flag when subscribing to events. If the flag is set, the widget will be notified immediately of any prior events of the type it is registered to receive.

Parameters

eventName String The name of the event to subscribe to.
callBack function function that will respond to events sent from other components of the broadcasting app. it will be given the event object itself and the source of the event.
receivePastEvents (optional) Boolean a flag to indicate that all past instances of the registered event should be sent to registered listener. This will happen immediately upon registration.

Return Value

Number subscription id.

Examples

Availability

Since 1.25.0 and later.

static

unsubscribe( )

unsubscribe( )

Description

Unsubscribes from receiving further events. The id from the initial subscribe call is used to unsubscribe from furthers notifications.

Parameters

eventName String The name of the event to unsubscribe from.
subscription Number Id returned by Wix.PubSub.subscribe.

Examples

Availability

Since 1.25.0 and later.
