openSettingsDialog( )

openSettingsDialog( )

Description

Available only from Preview. Ends Preview and opens the Settings Dialog for a given component, potentially navigating the user to a different page in the editor. When called with no options, uses the widget or page that called the function.

Parameters

options (optional) Object options may contain compId to open the settings panel for. If the provided component id is incorrect, the failure callback will be invoked.
failure (optional) function The callback is invoked either when the user cancels opening the settings panel, or if the specified component information is incorrect.

Examples

Availability

Since 1.45.0 and later.

Warning

This feature is experimental and it's behavior is not guaranteed. Not for use in production code.
